# Visualizations interactives

Explorez les concepts cles de React Native de maniere visuelle et interactive.

## Disponibles

- [Architecture React Native](./react-native-architecture.html) — Diagramme interactif des layers (JS thread, Bridge/JSI, Native thread). Comparez ancienne et nouvelle architecture.

- [Flexbox Playground](./flexbox-playground.html) — Bac a sable interactif pour experimenter flexDirection, justifyContent, alignItems, gap et plus.

- [Navigation Flow](./navigation-flow.html) — Visualisez les transitions entre ecrans : stack, tabs, drawer. Voyez l'etat de navigation en temps reel.

- [State Management](./state-management-comparison.html) — Comparaison cote a cote de Context, Zustand et Redux. Compteurs de re-renders interactifs.

- [Courbes d'animation](./animation-curves.html) — Visualisez les courbes timing, spring et decay. Ajustez les parametres et comparez.

- [Bridge vs JSI](./bridge-vs-jsi.html) — Animation du flux de donnees : Bridge (JSON serialise, async) vs JSI (acces memoire direct, sync).

- [Bundle Analyzer](./bundle-analyzer.html) — Treemap interactif d'un bundle React Native. Impact de Hermes, tree-shaking, taille des dependances.

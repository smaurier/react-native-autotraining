# Lab 07 — Composants UI avances

## Objectifs

Maitriser la creation de themes, les files de toasts, les proprietes d'accessibilite, le pattern compound components, le calcul de hitSlop, les badges et les avatars en TypeScript pur.

## Exercices

### Exercice 1 : createTheme
Creez un objet ThemeTokens complet a partir de couleurs fournies.

### Exercice 2 : mergeThemes
Fusionnez deux themes avec une strategie de merge profond.

### Exercice 3 : createToastQueue
Implementez une file de toasts avec un nombre maximum visible, ajout, suppression et vidage.

### Exercice 4 : createAccessibilityProps
Creez un objet de proprietes d'accessibilite React Native avec validation.

### Exercice 5 : buildCompoundComponent
Construisez un arbre de composants representant le pattern compound components.

### Exercice 6 : computeHitSlop
Calculez le hitSlop necessaire pour atteindre une taille cible tactile minimale.

### Exercice 7 : formatBadge
Determinez l'affichage et la visibilite d'un badge en fonction de son compteur.

### Exercice 8 : getKeyboardAvoidingBehavior
Retournez le behavior recommande de KeyboardAvoidingView par plateforme.

### Exercice 9 : createChipState
Gerez l'etat d'un groupe de chips avec selection multiple (toggle, clear).

### Exercice 10 : buildAvatarInitials
Generez les initiales d'un avatar a partir d'un nom complet.

Quiz 05 — StyleSheet et Flexbox

1. Quelle est la valeur par defaut de flexDirection dans React Native ?

Correct ! Contrairement au CSS web ou le defaut est row, React Native utilise column par defaut. Les enfants s'empilent donc verticalement.
Incorrect. En React Native, flexDirection vaut column par defaut (et non row comme en CSS web). Les enfants s'empilent verticalement.

2. Que signifie flex: 1 dans React Native ?

Correct ! flex: 1 (avec N > 0) equivaut a flexGrow: 1, flexShrink: 1, flexBasis: 0. L'element prend tout l'espace restant. Avec flex: 2, il en prendrait le double d'un element flex: 1.
Incorrect. flex: 1 equivaut a flexGrow: 1, flexShrink: 1, flexBasis: 0. L'element occupe tout l'espace disponible dans son conteneur. C'est le pattern le plus courant pour remplir l'ecran.

3. Comment fonctionne position: 'absolute' dans React Native par rapport au CSS ?

Correct ! En React Native, un element absolute est toujours positionne par rapport a son parent direct. Il n'y a pas de remontee de l'arbre pour trouver un ancetre positionne. position: fixed n'existe pas non plus.
Incorrect. En React Native, position: absolute positionne toujours l'element par rapport a son parent direct, sans chercher un ancetre avec position: relative comme en CSS.

4. Comment composer plusieurs styles de maniere conditionnelle dans React Native ?

Correct ! React Native accepte un tableau de styles. Les valeurs false, undefined et null sont ignorees. Le dernier style defini gagne en cas de conflit (strategie "last-wins").
Incorrect. On compose les styles avec un tableau : style={[styles.base, isActive && styles.active]}. Les valeurs falsy sont ignorees et le dernier style gagne en cas de conflit.

5. Comment creer un layout responsif qui s'adapte a la taille de l'ecran dans React Native ?

Correct ! useWindowDimensions retourne la largeur et la hauteur de la fenetre et se met a jour automatiquement (rotation, redimensionnement). On peut ensuite calculer le nombre de colonnes, les tailles et les marges en fonction.
Incorrect. React Native n'a pas de media queries CSS. On utilise le hook useWindowDimensions qui fournit la taille de l'ecran et se met a jour automatiquement lors des changements (rotation, etc.).
